# ChemAgent LoRA 微调训练报告

**项目：** 化工行业 5 岗位 AI Agent
**基座模型：** Qwen2.5-7B-Instruct
**微调方法：** LoRA (Low-Rank Adaptation)
**训练环境：** NVIDIA RTX 5090 D (24GB) · WSL2 Ubuntu · PyTorch 2.13+cu130
**训练时间：** 2026-06-14

---

## 一、训练数据

### 数据来源
| 来源 | 数量 | 说明 |
|------|------|------|
| NIOSH 安全数据模板 | 5,000条 | 105种化学品 × 5岗位 × 10模板问题 |
| DeepSeek API 生成 | 737条 | AI 生成的真实化工场景问答 |
| **合计** | **5,737条** | 训练4,626条 / 验证1,111条 |

### 数据分布
| 岗位 | 训练集 | 验证集 |
|------|--------|--------|
| 销售支持 (sales) | ~1,000 | ~150 |
| 配方研发 (formula) | ~1,000 | ~150 |
| 产线操作 (production) | ~1,000 | ~150 |
| 设备维修 (maintenance) | ~1,000 | ~150 |
| 质量检验 (qc) | ~1,000 | ~150 |

### 数据格式
每条样本为 ShareGPT 格式的对话对：
```
system: "你是 ChemAgent 化工行业 AI 助手..."
user: "客户说价格太贵怎么回？"
assistant: {"customer_intent": "...", "recommended_product": [...], ...}
```

## 七、后续优化方向

1. **数据增强** — 扩充化学品至 500+ 种，覆盖更多化工场景

5. **多模型集成** — 不同岗位使用不同的 LoRA adapter
